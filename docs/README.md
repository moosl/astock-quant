# A股 价值选股报告 · 网站说明

这是一个 **A股 量化研究学习项目** 的报告网站。它每隔一段时间跑一遍程序，
从沪深300里挑出「又便宜又能赚钱」的公司，生成一个网页报告。

> ⚠️ **最重要的一句话**：这是个**学习 / 研究**项目，所有内容**不构成任何投资建议**。
> 报告里说的「跑赢大盘」只是一段历史回测的结果，**不代表以后真能赚钱**。千万别拿它当炒股工具。

---

## 这个项目在做什么？（一句话版）

想象你要开一家小店收购别人的公司股票。你会想买什么样的？
当然是 **价钱便宜、又确实在赚钱** 的好公司。

这个项目就是让电脑替你干这件事：

1. 把沪深300里 300 家公司的财务数据全翻一遍；
2. 找出那些 **股价相对便宜**（市盈率、市净率低）、**又真的能赚钱**（净资产收益率高）的公司；
3. 给每家公司打一个「综合分」，分高的排前面，组成一份「推荐名单」；
4. 假设你照着这名单买入、**每个季度调整一次**、长期拿着，看看这样能不能跑赢大盘。

这套思路叫「**价值投资**」——巴菲特用的就是这一类方法。

## 重要：这个项目「换过方法」

这个项目 **2026-05-22 之前用的是另一套完全不同的方法**：

| | 旧方法（2026-05-22 之前） | 新方法（2026-05-22 起，现在用的） |
|---|---|---|
| 目标 | 猜一只股票**明天/下周涨还是跌** | 挑**又便宜又赚钱**的好公司长期持有 |
| 结果 | 被反复证明 **基本等于抛硬币**（准确率≈51%） | 一段历史回测里跑赢了大盘（详见下文，但口径要诚实） |
| 持有周期 | 短线，几天 | 中长期，按季度 |

**为什么要换？** 因为「预测短期涨跌」这件事，全世界的专业机构都做不好，
我们的旧模型也一样——它的准确率常年在 50% 上下，跟掷硬币没区别。
与其继续猜一件猜不准的事，不如改做「挑好公司」这件相对靠谱的事。

> 网站首页归档列表里，**早于 2026-05-22 的报告是旧方法生成的**，会标注「旧·涨跌预测」，

## 报告里每一段是什么意思

### ⚠️ 诚信声明
**这是每份报告最重要的一段**，开门见山把这套方法的局限讲清楚：
它只看历史财务数据、不预测短期涨跌、回测不代表未来、A股情绪波动大好公司也可能被错杀。

### §1 本季度价值选股推荐名单 ⭐
**这是报告的核心**。一张表，列出这一期电脑挑出的好公司，按「综合分」从高到低排：

| 这一列 | 怎么理解 |
|---|---|
| 综合分 | 把「便宜程度」和「赚钱质量」加权算出来的总分，**越高越好** |
| PE（市盈率） | 简单说：你出的价钱相当于公司**多少年的利润**。**越低越便宜** |
| PB（市净率） | 你出的价钱相当于公司**净资产的几倍**。**越低越便宜** |
| ROE（净资产收益率） | 公司用股东的钱**赚钱的效率**。**越高说明越能赚钱** |
| 入选理由 | 一句话说明这家公司为什么被选中（便宜？质量高？还是成长好？）|

打个比方：**PE/PB 像是看「这东西卖得贵不贵」，ROE 像是看「这东西本身好不好用」。**
价值选股就是想找「东西好、价钱还不贵」的那种。

### §2 策略回测表现 vs 沪深300
「回测」= 拿过去几年的真实历史数据，**假装**当时就照这个名单操作，算算能赚多少。
这一段把策略的成绩和大盘（沪深300指数）摆在一起对比：累计收益、超额收益、最大回撤等。

**诚实地说这段成绩**：在一段约 4 年（2022–2026）的回测里，这套策略累计涨了约 **+20.73%**，
同期沪深300 反而跌了约 **−7.96%**，年化超额约 **+6.25%**，5 年里赢了 4 年、输了 1 年。

**但请务必注意——「回测赢了」绝不等于「以后能赚钱」**：
- 样本太小：只测了 4 年、一种市场环境，换一段历史结果可能完全不同；
- 回测偏乐观：选股池是「今天的沪深300成分股」，这本身就是「涨上来的赢家」，退市的输家不在里面；
- 超额主要靠 2024 年单独一年撑起来的，其它年份优势并不明显；
- 衡量「稳不稳」的信息比率（IR）只有 0.46，偏低——说明这点超额并不算很稳。

所以这段成绩的正确读法是：「**这套方法在历史上没有被证伪、有一点跑赢的迹象**」，
而不是「这套方法保证赚钱」。

## 常见疑问

**Q：能照着 §1 的推荐名单买股票吗？**
A：**不建议**。这是一个学习 / 研究项目，目的是「学量化怎么做」，不是「给你荐股」。
回测成绩偏乐观、样本也小，照着实盘操作风险自负。

**Q：回测都跑赢大盘了，为什么还说不能信？**
A：回测是「拿历史数据假装操作」，它有几个天然缺陷（样本小、幸存者偏差、靠单年）。
全世界绝大多数「回测很漂亮」的策略，一到实盘就不行了。诚实承认这一点，才是这个项目的底线。

**Q：为什么旧报告还留着？**
A：留作历史记录，方便对比这个项目「方法是怎么演进的」。它们用的是已废弃的旧方法，
不代表现在的水平。

**Q：报告多久更新一次？**
A：价值选股是按季度调仓的策略，本身不需要天天看。报告会不定期重新生成。
