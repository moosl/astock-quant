# A股 每日预测报告 · 网站说明

这是一个 **A股 量化预测学习项目** 的报告网站。每天自动跑一遍模型，把预测结果生成一个网页。

> ⚠️ **最重要的一句话**：这是个**学习 / 研究**项目，所有预测**不构成任何投资建议**。模型本身没有「赚钱能力」（详见下面的诚信声明），千万别拿它当炒股工具。

---

## 怎么看这个网站

打开网址后，你会看到：

1. **首页**（`index.html`）—— 一个报告列表，最上面是今天的，往下是历史记录。每行显示当天的两个核心指标，点「查看 →」进入当天完整报告。
2. **每日报告** —— 点进去后是当天的详细预测，分成 7 个部分（下面逐个解释）。

---

## 报告里每一段是什么意思

每份每日报告分 7 段，从上到下：

### 🎯 今日一句话总结
最上面的速读区。如果模型今天「退化」了（坏掉了），这里会有一条**红色警告**，告诉你今天的结果不可信 —— 看到红警告就别往下细看了。

### ⚠️ 诚信声明
**这是本项目最重要的一段**。一个表格列出 4 个模型今天的「成绩单」：

| 指标 | 怎么理解 |
|---|---|
| AUC | 越接近 1 越好，0.5 = 抛硬币。**我们的模型常年在 0.5 上下** —— 意思是猜涨跌跟掷硬币差不多 |
| R² | 越接近 1 越好，负数 = 比「直接猜平均值」还差 |
| rank-IC | 选股排序能力，接近 0 = 排名跟抓阄一样 |
| macro-F1 | 买卖信号的综合准确率 |

**为什么要故意暴露自己很差？** 因为这是学习项目的「诚信红线」—— 量化模型 99% 都跑不赢市场，假装很厉害是骗自己。把真实（很差的）成绩摆出来，才知道差在哪、怎么改进。

### §1 今日总览
4 个模型今天有没有正常跑完。

### §2 ① 涨跌方向
预测每只股票 5 天后是涨还是跌。**注意**：这是个「二选一」模型，只有「涨 / 跌」两个答案，**没有「不变」**。如果你看到「300 只全部看跌」，多半是模型退化了 —— 看 §1 上面有没有红警告。

### §3 ② 预期收益率
预测每只股票未来涨跌幅百分比。

### §4 ③ 横截面排名
给所有股票按「看好程度」排个序，Top N 是模型最看好的。

### §5 ④ 买卖点信号
**这是目前最有参考价值的一段**。三分类模型：
- **TP**（止盈 / 建议买入观察）
- **HOLD**（持有 / 观望）
- **SL**（止损 / 建议卖出观察）

它比 §2 的「涨跌方向」更靠谱，因为任务更具体。

### §6 历史准确率
模型过去的预测事后对没对（数据积累中）。

### §7 运行元数据
技术信息：跑了多久、用了哪个模型文件等。

## 常见疑问

**Q：为什么每天都是「300 只全看跌」或「全看涨」？**
A：这是 §2 模型「退化」的表现 —— 它训练时找不到有用的规律，就对所有股票给了几乎一样的分。报告顶部的红色警告就是提示这个。这种时候看 §5（买卖点信号）更有意义。

**Q：能照着这个买股票吗？**
A：**不能**。模型 AUC ≈ 0.5，等于抛硬币。这个项目的目的是「学量化怎么做」，不是「给荐股」。

**Q：报告多久更新一次？**
A：每个交易日早上 8:00 和下午 16:30 各跑一次（取决于电脑有没有开机联网）。
